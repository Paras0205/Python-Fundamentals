# Dictionaries in Python

## Overview

A dictionary is a built-in Python data structure used to store data in **key-value pairs**. Dictionaries are mutable, unordered (prior to Python 3.7), and provide fast access to values using unique keys.

This module covers the fundamentals of Python dictionaries, including creation, manipulation, dictionary methods, nested dictionaries, looping through dictionaries, and practical examples.

# File Handling & Exception Handling in Python

## Overview

This module covers two essential Python concepts: **File Handling** and **Exception Handling**. These concepts are important for reading and writing data to files, managing resources efficiently, and handling runtime errors gracefully.

---

## Topics Covered

### 1. File Handling

Learn how to work with files in Python.

**Concepts Covered:**

* Opening files using `open()`
* Reading files
* Writing to files
* Appending data to files
* Closing files
* Using the `with` statement
* File modes (`r`, `w`, `a`, `x`)

---

### 2. Exception Handling

Learn how to handle errors without crashing the program.

**Concepts Covered:**

* `try`
* `except`
* `else`
* `finally`
* Handling common exceptions
* Raising custom exceptions (`raise`) *(if covered)*

---

## Skills Practiced

* Reading data from files
* Writing and appending data to files
* Managing files safely using context managers (`with`)
* Handling exceptions gracefully
* Preventing program crashes caused by runtime errors
* Writing robust and reliable Python programs

---

## Learning Outcomes

After completing this module, I learned:

* How to create, read, write, and append files in Python.
* The purpose of different file modes.
* How to use the `with` statement for automatic file management.
* How to identify and handle runtime exceptions.
* How exception handling improves the reliability of Python applications.
